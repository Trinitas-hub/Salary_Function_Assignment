Assignment for Salary Function

Files that are included
- module2_salary_function.ipynb (notebook for Python)
- Employee Profile.zip (a zipped folder with an exported employee CSV file)
- unzip_display_employee.R (R script to unzip and show CSV)
- README.md

How to Use (Python)
1. Put archive.zip into Jupyter Notebook.
2. Go through the cells in the notebook from top to bottom.
3. The notebook gets Total.csv, looks for an employee by name, exports their information to CSV, and makes Employee Profile.zip.

How to Run (R)
1. Open RStudio and set the working directory to the folder where Employee Profile.zip is stored. To do this, go to Session > Set Working Directory > Choose Directory.
2. Execute unzip_display_employee.R.
3. The script opens Employee Profile.zip and prints out the information about the employee.

Output
- Employee Profile.zip with the file employee_details_CHRISTOPHER_CHONG.csv
